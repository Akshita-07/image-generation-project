# Image Generation using Stable Diffusion

## Description
This project generates images from text prompts using a pre-trained Stable Diffusion model.

## Tools Used
- Python
- Google Colab
- Diffusers Library

## Output
The model generates high-quality images based on input prompts.

## Example Prompt
"A futuristic city with neon lights at night"
